This library is intended to provide some light language toolkit / data mining functions built on top of pattern clips 

http://www.clips.ua.ac.be/pages/pattern-en

It is a javascript front end with python on the back

Follow instructions for setup here: https://community.c9.io/t/running-a-python-cgi-server/1602

Everytime you add a file to cgi-bin you must run this script: 
    
    chmod +x -R /home/ubuntu/workspace/cgi-bin
